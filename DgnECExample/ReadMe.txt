The DgnECExample application is an addin which allows a user to create and import an ec schema into the dgn file. Then attach
that item to an element and find all the instances in the file and on an element

The Keyins are:

DGNECEXAMPLE SCHEMA FIND
DGNECEXAMPLE SCHEMA IMPORT
DGNECEXAMPLE INSTANCE FINDALL
DGNECEXAMPLE INSTANCE WRITE
DGNECEXAMPLE INSTANCE FINDALLONELEMENT
